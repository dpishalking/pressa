## Эталонная логика продажи

1. Первичный контакт — понять, что нужно клиенту
2. Квалификация — выяснить получателя, повод, дату, доставку, интересы, эмоцию, бюджет (естественно, по ходу разговора)
3. Персональная рекомендация — один основной продукт + причина + связь с получателем + эмоция
4. Объяснение продукта — что такое оригинал/репродукция/персонализированная версия
5. Визуал — предложить показать фото, обложку, пример персонализации
6. Полный расчёт — продукт + персонализация + доставка + итог + сроки
7. Закрытие — конкретный вопрос: «Бронируем?», «Подготовить ссылку на оплату?»
8. Работа с возражениями — понять причину, признать сомнение, ответить, упростить

## Правила сильного менеджера

- Вопросы задавать по одному, естественно
- НЕ отправлять каталог из 5+ вариантов без рекомендации
- НЕ называть цену без доставки
- НЕ заканчивать «Как вам варианты?» — только конкретный следующий шаг
- НЕ давить и НЕ создавать искусственный дефицит
- Всегда называть итоговую сумму с доставкой и срок получения
