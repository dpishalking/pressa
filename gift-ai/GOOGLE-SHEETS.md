# Каталог подарков из Google Таблицы

Бот **не придумывает** подарки — он читает только то, что в таблице.

---

## 1. Формат таблицы

**Первая строка — заголовки.** Пример:

| id | название | описание | цена | эмоции | кому | поводы | срок | персонализация | фото | кейсы | отзывы | активен |
|----|----------|----------|------|--------|------|--------|------|----------------|------|-------|--------|---------|
| gazeta-01 | Персональная газета | Газета со статьей о человеке | 3500-8900 | удивление, любовь | папа, мама | день рождения, юбилей | 5 | имя, факты, фото | https://... | Семья плакала от счастья | Отличный подарок | да |
| portret-02 | Портрет маслом | ... | 15000-45000 | ... | ... | ... | 14 | ... | | | | да |

### Колонки (русские или английские названия)

| Колонка | Обязательно | Пример |
|---------|-------------|--------|
| `id` / `артикул` | желательно | `gazeta-01` |
| `название` | **да** | Персональная газета |
| `описание` | да | Текст для бота |
| `цена` или `цена_от` + `цена_до` | да | `3500-8900` или `5000` |
| `эмоции` | да | через запятую |
| `кому` | да | папа, мама, жена |
| `поводы` | да | день рождения, юбилей |
| `срок` | нет | дней на изготовление |
| `персонализация` | нет | что можно настроить |
| `фото` | нет | ссылка на картинку |
| `кейсы` | нет | пример заказа |
| `отзывы` | нет | цитата клиента |
| `активен` | нет | `да` / `нет` |

---

## 2. Открыть доступ к таблице

**Файл → Настройки доступа** → «Все, у кого есть ссылка» → **Читатель**

Или: **Файл → Поделиться → Опубликовать в интернете** (лист с каталогом).

---

## 3. Получить ссылку для бота

Из URL таблицы:
```
https://docs.google.com/spreadsheets/d/ВОТ_ЭТОТ_ID/edit#gid=0
```

Ссылка для бота (CSV):
```
https://docs.google.com/spreadsheets/d/ВОТ_ЭТОТ_ID/export?format=csv&gid=0
```

`gid=0` — номер листа (вкладки). Если каталог на другом листе — откройте лист и скопируйте `gid` из URL.

---

## 4. Подключить на Railway

Сервис **pressa** (API) → **Variables**:

```
GOOGLE_SHEET_CSV_URL = https://docs.google.com/spreadsheets/d/.../export?format=csv&gid=0
```

**Redeploy** — при старте каталог подтянется из таблицы.

Проверка: `/health` → `"gifts": 12` (число ваших подарков).

---

## 5. Обновить каталог

Изменили таблицу → снова **Redeploy** API  
или вручную:

```bash
curl -X POST https://pressa-production.up.railway.app/admin/sync-sheets \
  -H "x-admin-key: ВАШ_ADMIN_API_KEY"
```

---

## Советы

- Чем **подробнее описание** — тем точнее рекомендации бота
- **Эмоции** и **кому** — главное для подбора
- `id` не меняйте — по нему обновляются строки при синхронизации
- `активен` = `нет` — подарок скрыт, бот его не предлагает
