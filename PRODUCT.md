# Pressa — продукт

## One-liner

Pressa — vibe-coding для **газетных статей**: описал сюжет и настроение → получил структурированный черновик в редакционном формате.

## Для кого

- Региональные и онлайн-газеты
- Фриланс-журналисты
- Редакции без большого штата копирайтеров
- Студенты журфака (черновик + обучение структуре)

## Core flow (MVP)

1. **Тема** — о чём материал, факты, источники (кратко)
2. **Рубрика** — новость / репортаж / интервью / мнение / очерк
3. **Тон (vibe)** — нейтрально-информационный, острый, тёплый, деловой…
4. **Выход** — headline + lead + body (+ опционально: подзаголовки, блок «в двух словах»)

## Не делаем на старте

- Вёрстку полосы и PDF газеты
- Автопубликацию без проверки человеком
- Выдумывание фактов без пометки «проверить»

## Монетизация (гипотеза)

- Free: N статей / месяц
- Pro: без лимита, стили редакции, экспорт Word/Google Docs
- Redacción: несколько авторов, гайдлайн редакции
