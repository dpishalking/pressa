# 001 — Nombre Pressa

**Fecha:** 2026-05-30  
**Estado:** Aceptado

## Decisión

El proyecto se llama **Pressa** (carpeta `pressa`, marca Pressa).

## Motivo

- Evoca **prensa** / press sin ser genérico «News AI»
- Corto, memorable para LATAM y Europa
- Encaja con vibe-coding editorial, no con otro vertical (ej. cotizaciones)

## Alcance

Servicio de ayuda a **textos para artículos de periódico**, no redes sociales ni blogs largos SEO (por ahora).
