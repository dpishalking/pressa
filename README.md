# Pressa

**Vibe-coding сервис** для создания текстов **газетных статей** — от заметки до готового материала в редакционном стиле.

> Код приложения (когда появится) живёт в этом репозитории или в отдельной папке `app/`.  
> Здесь — продукт, процессы, шаблоны и решения.

## Идея

Помочь журналистам, редакциям и авторам быстро получить **черновик статьи** с правильной структурой газеты:

- заголовок (headline)
- лид (lead / entradilla)
- тело с абзацами
- подзаголовки, цитаты, справка

Пользователь описывает тему и тон («вайб») — Pressa собирает текст, который можно доработать вручную.

## Структура репозитория

| Папка | Назначение |
|-------|------------|
| `procesos/producto/` | Roadmap, фичи, UX вайбкодинга |
| `procesos/editorial/` | Стиль, рубрики, длина, этика |
| `procesos/contenido/` | Типы материалов, чеклисты |
| `plantillas/` | Шаблоны статей и промптов |
| `decisiones/` | Журнал продуктовых решений |
