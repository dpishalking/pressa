# Rúbricas soportadas (MVP)

| Rúbrica | Longitud ref. | Salida |
|---------|---------------|--------|
| **Noticia** | 300–600 palabras | Titular + lead + 4–6 párrafos |
| **Reportaje** | 800–1500 | Titular + lead + bloques con subtítulos |
| **Entrevista** | 600–1000 | Presentación + Q&A o narrativa |
| **Opinión** | 500–800 | Tesis en lead + argumentos |
| **Breve** | 150–250 | Titular + 2–3 párrafos |

## Campos que pide el usuario (vibe form)

- Tema / ángulo
- Hechos conocidos (bullets)
- Fuente o contexto (opcional)
- Ciudad / audiencia (opcional)
- Tono: slider o etiquetas
- Extensión: corta / media / larga
